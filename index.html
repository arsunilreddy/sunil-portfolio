🚀 Current Project

Wrist Vein Scanner Biometric System

I am developing a real-time biometric authentication system that captures and verifies wrist vein patterns using near-infrared (NIR) imaging. The system is deployed on the NVIDIA Jetson Orin Nano, supporting both battery and power supply operation for flexible deployment in mobile or stationary environments.

It includes a custom PCB interface with an OLED display and physical controls, and uses techniques like HDR image fusion, CLAHE, and hair removal to enhance vein visibility. Feature extraction is performed using a ResNet50-ArcFace model, and shallow neural networks (SNNs) are trained to generate 32-bit biometric keys, verified via SHA256 hashing and Hamming distance comparison.
